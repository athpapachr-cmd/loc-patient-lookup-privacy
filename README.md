# Privacy Policy

LOC Patient Lookup is used exclusively by Limassol Orthopaedic Center staff.

The extension reads a phone number entered in Cal.eu and securely sends it to the clinic backend for patient identification purposes.

No data is sold, shared with third parties, or used for advertising.

The extension stores the backend URL and API secret locally in the browser extension storage.

Patient information is used solely for appointment booking and patient identification within the clinic workflow.
